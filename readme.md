# Dictatorbot
A Twitch chat bot that keeps track of how many times a user has had it's messages deleted in chat.

## Installation
`npm install`

## Run the application

### In production
```npm run start```

### For development
```npm run dev```

#### Linting
```npm run lint```

#### Tests (coverage)
```npm run test```

#### Tests (autorun)
```npm run test-watch```




## Technologies
* Nodejs / CommonJS
* Jest
